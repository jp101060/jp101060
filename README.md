Engenheiro Mecânico (Senai Cimatec, semestres finais) que descobriu dados e não parou mais.

Comecei como estagiário de produção de conteúdo — e nos últimos 10 meses construí, do zero, ferramentas analíticas que hoje são usadas em produção real pela equipe: dashboards conectados ao banco de dados corporativo, uma API com visão computacional que venceu um hackathon interno, e relatórios gerenciais em Power BI que apoiam decisões do dia a dia.

## Projetos em destaque

### [📊 Production Analytics Dashboard](https://github.com/jp101060/production-analytics-dashboard)

Dashboard interativo de eficiência operacional conectado a banco de dados corporativo em produção. Análise individual por colaborador com detecção de anomalias via **Z-score**, médias móveis de 7 dias, heatmap de intensidade e análise temporal por segundo.

> O Z-score individual normaliza cada pessoa pelo próprio histórico — mais justo que comparar todos pelo mesmo padrão global.

`Python` `Streamlit` `Plotly` `Pandas` `SQL Server` `Estatística`

---

### [⚡ Relé-volution — Electrical Diagram Digitizer](https://github.com/jp101060/electrical-diagram-digitizer)

🏆 **Vencedor de hackathon interno**

API que converte fotografias de caixas de fusíveis e relés em diagramas vetoriais SVG padronizados usando visão computacional. O modelo identifica cada componente, lê o label via OCR e posiciona tudo sem sobreposições — em segundos.

> 🔗 API ao vivo: [meu-backend-ia-zats.onrender.com](https://meu-backend-ia-zats.onrender.com)

`Python` `FastAPI` `Gemini Vision` `LangChain` `Pydantic` `SVG` `Render`
